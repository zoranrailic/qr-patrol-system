Basic Usage

How to get the Key?

1) Visit your website where the system has been installed

2) (Login to your page)

3) (Go to -> Settings page)

4) As you can see the Key on this page

5) Don't forget to add KEY to your application (scan the QR)

How to get my employee or student attendance data?

1) Visit your website where the system has been installed

2) (Login to your page)

3) (Dashboard -> Attendance page)

*Note: You can search for your employees or students through filters, for example, searches by specific names or dates.

How to generate the QR code for attending?

1) Visit your website where the system has been installed

2) (Login to your page)

3) (Dashboard -> History QR page)

4) Fill in your employee or student name

5) Click Add

Can I add a new user login?

Yes, you can.

To do it please follow the instruction.

1) Visit your website where the system has been installed

2) (Login to your page)

3) (Dashboard -> Users)

4) Click "Add or Update"

How to import from CSV?

1) Visit your website where the system has been installed

2) (Login to your page)

3) (Go to -> Users -> Import CSV)

4) Please download first the CSV template

5) To add data please follow the instructions by clicking the "Show Instructions"

6) upload the CSV file

7) Click import

8) Done

*Important: Please pay attention, this system has 2 types of users. The first user with login means this user can attend and also login to system web, the second user without login means this user only attend and cannot login to the system. This CSV only work with role 2 and 3 (admin, and staff).

How to fill the purchase code?

1) Visit your website where the system has been installed

2) (Login to your page)

3) If you never fill the purchase code, the message will show up, and please click the link to fill in the purchase code.

4) Fill the purchase code

5) Done

*Important: You have 7 days to fill the purchase code) If you buy from Codecanyon you can find the purchase here
